# Plex Stack

* **Prowlarr** manages what indexers are searched
  * **Flaresolverr** attempts to solve Cloudflare anti-automation challenges
* **Radarr** manages available movies
* **Sonarr** manages available shows
* **Seerr** provides a user-friendly interface to search and request titles, then sends those requests to the applicable tool
* **Tautulli** monitors Plex activity and provides reports and notifications
